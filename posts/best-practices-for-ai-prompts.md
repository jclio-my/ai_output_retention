# AI 提示词最佳实践

与 AI 交互时，一个好的提示词往往能获得更好的结果。以下是一些实用的技巧。

## 1. 明确目标

清楚地告诉 AI 你想要什么：

> "请帮我写一个 Python 函数，用于计算斐波那契数列"

比

> "帮我写个函数"

更有效。

## 2. 提供上下文

给 AI 足够的背景信息：

```
我是一名前端开发者，正在学习 React。
请解释一下 useState 的工作原理，并给出一个简单的例子。
```

## 3. 指定格式

如果你需要特定格式的输出，明确说明：

> "请以 JSON 格式返回结果"
> "请用表格形式对比这两种方法的优劣"

## 4. 迭代优化

如果第一次结果不理想，继续追问和调整：

- "能不能更简洁一点？"
- "请添加更多注释"
- "这个方法的缺点是什么？"

## 5. 举例说明

提供例子可以让 AI 更好地理解你的需求：

```
我想要这样的效果：
输入: "hello world"
输出: "HELLO WORLD"

请帮我实现这个函数
```

---

记住，AI 是你的协作伙伴，清晰的沟通能带来更好的协作效果！
